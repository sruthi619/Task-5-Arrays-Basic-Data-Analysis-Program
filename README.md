# Task-5-Arrays-Basic-Data-Analysis-Program
## Objective
To analyze numerical data using Java arrays and perform operations like sum, average, min, max, and sorting.

## Tools Used
- jdk-25
- Java 8+

## Concepts Covered
- Single & Multi-dimensional arrays
- Loops and traversal
- Manual sorting (Bubble Sort)
- Exception handling
- Formatted console output

## Outcome
Understood how arrays store data and how to analyze datasets using core Java.
